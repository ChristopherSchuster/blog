El circuito

Unir una resistenica de 220-ohm resistor(bordes rojo y dorado) al el pin 13. Luego unir la pata mas larga de led (pata positiva, anodo) a la resistencia. Unir la pata mas corta (negativa, catodo) a ground.

El codigo

Abrir el IDE Arduino y pegar el siguiente codigo 

const int LED = 13; // LED connected to
                    // digital pin 13

void setup()
{
  pinMode(LED, OUTPUT);    // sets the digital
                           // pin as output
}


/**1. Turns on the led
   2. Wait one second
   3. Tuns off the led
   4. Wait one second
**/
void loop()
{
  digitalWrite(LED, HIGH); 
  delay(1000);               
  digitalWrite(LED, LOW);  
  delay(1000);              
}

Compilar y subir el codigo a la tarjeta arduino.

Hola mundo en codigo morse

El codigo morse ha sido usado por añs para comunicarse entre largas distancias, vamos a tratar de programar el arduino para que diga hello world en codigo morse:

  • H: ....
  • E: .
  • L: .---
  • O: ---
  • W: .--
  • R: .-.
  • D: -.. 
.... . .--- .--- ---  .-- --- .-. .--- -..

Por cada raya el led se iluminar´ 1.6 segundos, por cada punto 800 milisegundos, el tiempo de espera entre letras es 500 milisegundos y entre palabras 1.5 segundos
